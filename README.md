## Overview
This is an ASP.NET MVC web application developed to demonstrate API integration and asynchronous programming in C#. The application fetches jokes from an external service and displays them by category (e.g., Programming, Puns).

## Features
* **MVC Architecture:** Built using Model-View-Controller patterns.
* **Async API Calls:** Uses `HttpClient` to fetch data without freezing the UI.
* **Dynamic Categories:** Displays jokes such as "Programming" and "Puns" with setup and delivery.

## Tech Stack
* **Framework:** ASP.NET MVC (.NET Framework)
* **Language:** C#
* **Frontend:** Razor Views (HTML/CSS)
* **Tools:** Visual Studio 2022
